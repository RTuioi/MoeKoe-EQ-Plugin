
MoeKoe EQ 是一款专为 MoeKoeMusic 音乐播放器设计的31 段参数均衡器 Chrome 扩展插件。提供专业的31段参数均衡器和的音效增强功能，让用户获得更好的音频体验。

                    首先十分感谢“ 阿珏 ”开发的  MoeKoeMusic   项目,并且感谢他长久以来对项目的维护和优化。


-------------------------------------------------
                    

                                                🎵 MoeKoe EQ - 31段均衡器音效插件
                                  

### 🎛️ 核心功能 1. 31段参数均衡器
- 频率范围 : 20Hz - 20kHz，覆盖完整可听频段
- 增益范围 : -6dB 到 +6dB，每档 0.5dB
- 频率点 : 20, 25, 31.5, 40, 50, 63, 80, 100, 125, 160, 200, 250, 315, 400, 500, 630, 800, 1000, 1250, 1600, 2000, 2500, 3150, 4000, 5000, 6300, 8000, 10000, 12500, 16000, 20000 Hz 2. 内置预设
预设名称 适用场景 平坦 默认无修饰 摇滚 摇滚音乐 古典 古典音乐 流行 流行音乐 爵士 爵士乐 低音增强 重低音爱好者 高音增强 清晰高音 人声 人声突出 仿：风雪调音 特色调音 极致听感 综合增强 醇美空间 空间感增强
 3. 音效增强
- 频率调节 : 低频提升、低音增强、温暖感、人声增强、临场感、清晰度、高频提升、动态增强
- 空间效果 : 环境感、环绕声、环境混响
- 输出控制 : 输出增益、声道平衡

### 🔧 技术实现
文件 职责 background.js 设置存储管理、消息路由、预设管理 content.js UI 界面渲染、用户交互、Shadow DOM 隔离 inject.js 音频处理核心、AudioContext 管理、EQ 链路构建 popup.js 弹出窗口控制、快捷操作


###  UI 特性
- 悬浮按钮 (FAB) : 右下角快速访问
- 实时曲线显示 : Canvas 绘制 EQ 曲线
- Shadow DOM 隔离 : 避免样式冲突


### 使用方法
1. 安装插件后重启 MoeKoeMusic
- 首次安装后，点击托盘 → MoeKoeMusic → 重启应用

2. 点击页面右下角悬浮按钮打开面板
- 页面右下角会出现均衡器悬浮按钮

3. 点击页面右下角悬浮按钮打开面板
- 点击按钮打开控制面板

4. 开启音效增强获得更好体验
- 选择预设或手动调节各频段
- 可保存自定义预设供后续使用


### 注意事项
- 首次安装需重启应用
- 播放音频时自动激活

-说明一下，这个是用 AI 写的插件，可能会出现一些 bug，是否使用需自行考量。
