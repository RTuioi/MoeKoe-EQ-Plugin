# MoeKoe-EQ-Plugin
MoeKoe EQ 是一款专为 MoeKoeMusic 音乐播放器设计的31 段参数均衡器 Chrome 扩展插件。它采用 Web Audio API 实现，提的音频调节功能和多种预设效果，让用户获得更好的音频体验。



                                  🎵 MoeKoe EQ - 31段均衡器音效插件
                                  ## 🎵 MoeKoe EQ - 31段均衡器音效插件

### 功能特性
- **31段参数均衡器**：20Hz - 20kHz 全频段覆盖，精确调节每个频段
- **11种预设模式**：流行、摇滚、古典、爵士、电子、人声、低音增强、高音增强、自定义 等
- **10种音效增强**：
  - 低频提升 / 动态低音 / 温暖感
  - 清晰度 / 临场感 / 人声增强 / 高频提升
  - 动态增强 / 环境氛围 / 环绕声
- **输出控制**：输出增益、声道平衡、混响效果
- **设置持久化**：自动保存用户设置

### 使用方法
1. 安装插件后重启 MoeKoeMusic
2. 点击页面右下角悬浮按钮打开面板
3. 选择预设或手动调节均衡器
4. 开启音效增强获得更好体验

### 技术实现
- 基于 Web Audio API 实现
- 31个 BiquadFilter 节点串联
- DynamicsCompressor 动态压缩
- Convolver 混响效果

### 注意事项
- 首次安装需重启应用
- 播放音频时自动激活

-说明一下，这个是用 AI 写的插件，可能会出现一些 bug，是否使用需自行考量。
